Add an Item to Inventory: addItemToInventory('item_name');

Add Multiple Items at once: addItemToInventory('item_name'); addItemToInventory('item_name');

Set a Progress Flag: setFlag('mini_game_solved_flag', true);

Remove an Item: removeItemFromInventory('item_name');

Reset (Clear Everything): localStorage.clear(); -->  location.reload();

Jumping Scenes: currentSceneIndex = 25; // Change 25 to whatever number you want   processScene();
